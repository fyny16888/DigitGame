参数 无
返回：
{
   code:200

}
======================================
开始游戏：
to.toHandler.startGame
参数: 无
返回：
{
	code:200
   data:{poke:要放的牌,count:剩余牌数}
}
返回错误 当前游戏次数不足


======================================
放牌：---------------------------------------------------------------------------------------------------------修改。
to.toHandler.setCard
参数 channel : （0 1 2 3 ）道
返回：
{
   code:200
   //data:{poke:要放的牌,count:剩余牌数,toScore:21点总积分,channel:哪个道,channelCount:渠道分数,boom:0 未爆牌 1 爆牌 2消除}
   data:{
		poke: 下一次要放的牌,(无牌为0)
		count：剩余牌数 , (无牌为0)
		cScore: [0,1,2,3]， 分别为4个道的当前计分
		toScore: 当前玩家总分数，
		boom： 0，无操作    1，爆牌    2，消除
		}
}

////
ps：无牌时完成游戏
返回：{
   code:200
   data:{toScore:21点总积分}
}
